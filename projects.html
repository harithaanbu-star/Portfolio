VenomGuard AI

SNAKEBITE EMERGENCY PREDICTION SYSTEM

An AI-powered snakebite emergency tool, built after a real 2 a.m. emergency call. It identifies the snake species from a photo or name using the Google Gemini Vision API, then predicts a personalised treatment window and antivenom estimate with a Gradient Boosting ML model — trained across 40 snake species on roughly 8,000 images, reaching ~91% accuracy. A 3-step workflow (species ID → patient details → AI prediction) outputs the exact time-critical window a victim has before irreversible damage, escalating instantly to a CRITICAL alert for high-risk species.

Hospital Staffing Demand Forecast

FORECASTING + SHIFT SCHEDULE OPTIMISATION

An applied data science project (AICTE-affiliated) that forecasts hospital patient demand using XGBoost, then feeds those forecasts into an OR-Tools optimisation engine to auto-generate efficient staff shift schedules. Paired with a FastAPI + React + PostgreSQL stack, with a real hospital data acquisition plan to validate the forecasts against ground truth.
